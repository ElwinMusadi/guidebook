Perkenalan

Perkenalan

Paket.id merupakan sebuah inovasi pengiriman barang dengan sistem booking dan pickup.

Fitur - fitur paket.id:

  • Instan booking, 5 digit kode pengganti detail pengirim dan penerima. Tidak menyita banyak waktu menulis alamat lengkap di paket.
  • Tracking, dapatkan informasi tracking untuk semua paket dalam satu platform melalui history booking.
  • SMS notifikasi, kami mengirim sms tracking link ke penerima secara gratis.
  • Layanan pickup
  • Laporan pengiriman, dapatkan laporan pengiriman Anda dengan mudah.
  • Integrasi dengan API, manfaatkan API pada website Anda untuk mengatur pengiriman.
  • Multivendor, gunakan kode booking di vendor pengiriman favorit Anda.
